### Règle de validation IEC
```
Si |calcul - référence| / référence < 1% → VALIDÉ
Si |calcul - référence| / référence < 5% → ACCEPTABLE
Si > 5% → ERREUR D'IMPLÉMENTATION
```

### Indicateur "Markov requis" (λ×T1)
```
λ_D × T1 < 0.05  → IEC approx. suffisante (erreur < 1%)
λ_D × T1 < 0.1   → IEC approx. acceptable (erreur < 5%)
λ_D × T1 > 0.1   → Markov exact RECOMMANDÉ
λ_D × T1 > 0.3   → Markov exact OBLIGATOIRE
```
